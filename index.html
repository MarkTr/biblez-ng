BibleZ NG

BibleZ NG is an web app for Firefox and Firefox OS. It is based on swordjs and EnyoJS.

The main goals are:

  • provide offline access to bible modules in many different languages (BibleZ NG is build upon the modules from CrossWire).
  • easy-to-use interface
  • great features like search, personal note and bookmarks/highlights, commentaries and more
How to build the app

Just run ./buildApp.sh from the root folder and you will have the app in the deploy/biblez-ng/ folder.
